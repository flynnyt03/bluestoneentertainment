GMod Fretta

The GMod Fretta 13 server is a dedicated server running the Fretta game mode updated to Garry's Mod 13 from Garry's Mod 9. Fretta is a versatile game mode that allows players to play a variety of different game types, including Pedobear Escape, Dodgeball, Suicide Barrels, and many others.

On the GMod Fretta 13 server, players can join in on the action and play against other players from around the world. The server is hosted on powerful hardware and features low latency, making for a smooth and enjoyable gaming experience.
